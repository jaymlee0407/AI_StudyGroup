## 🚀 Projects & Resources

**스터디 방식:**  
- 주 1회 정기 모임
- 선정해 온 자료를 함께 보며 공부하고 서로 질문하기
- 각 모임 후 발표 및 토론  
- GitHub를 통한 자료 및 코드 공유

**스터디 기록 작성방법:**  
- '작성 템플릿_스터디 기록.md'를 참고하여 작성하도록 한다.
- 해당 회차에 진행했던 내용을 요약하여 기록하고, 공부한 자료의 링크를 삽입한다.
- 다음 회차에 진행할 내용에 대해 기록하고, 함께 공부하고 싶은 자료의 링크를 삽입한다.
